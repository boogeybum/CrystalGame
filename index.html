Crystal Collector


You will be given a random target number at the start of the game.

There are four character pictures below. By clicking on a character you will add a specific amount of points to your total score.

You win the game by matching your total score the random target number. You lose the game if your score goes above the random target number.

The value of each character is hidden until you click on it.

Each time the game starts, a new random target number is selected and the values of each character will change.

Good Luck!
